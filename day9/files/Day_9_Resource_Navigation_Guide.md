## 💡 KEY CONCEPTS TO REMEMBER

### The Golden Rule of Prompts
```
Specific + Detailed + Examples (if complex) = Great Output
Vague + Generic + No examples = Poor Output
```

### Zero-Shot = Fast, Few-Shot = Better Quality
- Use zero-shot for simple, straightforward tasks
- Use few-shot for anything complex or requiring specific format

### The 6 Essential Elements
Every prompt should have:
1. Clear task
2. Target audience
3. Desired tone
4. Format requirements
5. Key elements to include
6. Constraints (what to avoid)

### The Iteration Cycle
Test → Evaluate → Refine → Test again → Perfect

## 🎯 EXAMPLE WORKFLOW

Let's say you want to create marketing copy for a new fitness app:

### Using Zero-Shot (Quick)
```
Write product copy for FitAI, an AI fitness app.
```
**Result:** Generic, okay but not great (⭐⭐⭐)

### Using Few-Shot (Better)
```
Write product description for FitAI following this example:

Example: "Your AI personal trainer learns your fitness level, goals, and equipment—then creates custom workouts with real-time form feedback and progress tracking."

For FitAI, create copy that:
- Targets busy professionals 25-45
- Emphasizes personalization and ease
- Addresses concern: "I don't know how to exercise"
- Length: 100-150 words
```
**Result:** Much better! Specific, targeted, compelling (⭐⭐⭐⭐⭐)

## 📞 TROUBLESHOOTING

**Problem: ChatGPT output is too generic**
→ Add more specifics: audience details, tone, examples

**Problem: Output is too long**
→ Specify word count explicitly (e.g., "exactly 150 words")

**Problem: Wrong tone**
→ Provide an example of the tone you want

**Problem: Missing key information**
→ List "must include:" section explicitly

**Problem: Format is wrong**
→ Show example of correct format

**Problem: Not actionable**
→ Specify the desired outcome clearly
