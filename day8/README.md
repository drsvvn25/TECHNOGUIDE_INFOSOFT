# Day 8 - ChatGPT API Integration Project

This project is a simple Python chatbot using the OpenAI API.

## Requirements

```bash
pip install -r requirements.txt
```

## Usage

1. Set your OpenAI API key in PowerShell:

```powershell
$env:OPENAI_API_KEY="your_openai_api_key_here"
```

2. Run the chatbot:

```powershell
python day8.py
```

3. Type a message and press Enter.
4. Type `exit` to stop the chat.

## Demo Mode

If you don't have an API key available, run the app without setting `OPENAI_API_KEY` and type `demo` when prompted. The app will switch to a built-in demo mode.
