## 💻 Major Projects

Here are the key projects developed during the internship, blending web tech with AI:

### 1️⃣ AI Resume Generator
* **Objective:** Simplify resume building using AI-driven content generation and ATS optimization.
* **Features:** Multiple dynamic templates, real-time preview, AI content suggestions, and PDF export.
* **Technologies Used:** HTML5/CSS3, JavaScript, Bootstrap, AI APIs.
* **Key Learning:** Implementing real-time DOM updates and connecting external LLM endpoints.

### 2️⃣ AI Invoice & Caption Generator
* **Objective:** Automate tedious business tasks for enhanced productivity.
* **Features:** Generate professional invoices, export as PDF, and craft engaging social media captions via AI.
* **Technologies Used:** Node.js, Express, HTML/CSS, REST API.
* **Key Learning:** Handling complex data entry, processing logic, and generating structured digital assets.

### 3️⃣ Modern Developer Portfolio Website
* **Objective:** Showcase skills, projects, and achievements in a sleek, responsive interface.
* **Features:** Dynamic 3D animations, responsive layouts, contact forms, and interactive UI components.
* **Technologies Used:** React, Tailwind CSS, Vite, Three.js.
* **Key Learning:** Advanced frontend aesthetics, CSS layout systems, and modern component structuring.

### 4️⃣ Final AI-Based Business Project
* **Objective:** A comprehensive application integrating various AI tools to streamline workflows.
* **Features:** Authentication systems, intelligent chatbot integration, user dashboards, and data visualization.
* **Technologies Used:** MERN Stack, Next.js, Google Gemini/OpenAI APIs.
* **Key Learning:** State management, full-stack deployment architectures, and secure API handling.
