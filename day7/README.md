# IPL Player Performance Analysis System

This project is a Streamlit app for IPL player performance analysis using machine learning and data visualization.

## Run locally

```bash
cd day7
pip install -r requirements.txt
streamlit run day7.py
```

## Deployment

- `requirements.txt` lists the Python dependencies.
- `day7.py` is the main Streamlit app source.
- `streamlit_app.py` is provided as an alternate Streamlit entrypoint.
- `cricket_data.csv` contains the player dataset used by the app.
